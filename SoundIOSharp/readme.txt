The bin_libs directory contains the binary native library for Windows and Linux. For Windows, each DLL for a specific 32 or 64 bit version is contained within the specific directory. The .NET binding library reads the library's app.config file to load the correct DLL at runtime. Note, only a 64-bit library is provided for Linux
